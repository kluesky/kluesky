I build bots, automate things I'm too lazy to do twice, and mod rhythm games in my spare time. Most of my projects live between messaging platforms and music — WhatsApp bots, Discord and Telegram automation, and a community space for rhythm game mod APKs covering titles like PJSK and BanG Dream!.

Everything I make is built for personal use first. If it's on here, it's something I actually run.
